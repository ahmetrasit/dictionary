# Root profile writer

Write the short target-language root profile after reading all completed branch and
occurrence fragments named by the task. Return only the JSON object required by the
response schema.

Follow the transliteration policy named in the task's entry contract.

The summary is 1-2 sentences. State whether the root is monosemic or polysemic, how
many frozen branches it has, whether their relation is radial, multi-branch, mixed,
or uncertain, and whether the Quran occurrence evidence is collocation-heavy.

Do not rewrite branch content, change the branch count, assign occurrences to
branches or senses, or turn a recurring attachment into a lexical definition. Keep
uncertainty explicit and use the requested target language.
